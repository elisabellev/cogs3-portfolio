my work

Study Buddy

This is the first webpage I've made from scratch. Study Buddy is a project that was assigned in my introductory coding class at UCSD, where I learned basic HTML and CSS skills. I also learned to navigate the Bootstrap 4 framework to complete this project.

My Teal Denim

My Teal Denim is a webpage I created with Wix before I learned how to code. The goal of the site is for young sexual assault survivors to find support and resources for their healing. The platform has been extended to Instagram, which is linked in the site.

Dream

Another project for my introductory coding class was to create an app prototype using FIGMA. The idea I had was "Dream", a sleep tracker app. This project allowed me to practice design and usability, as well as navigating FIGMA itself.
